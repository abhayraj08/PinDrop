# PinDrop
Click on the map, it will give the exact location. 

<h2>Key Feature</h2>
<ul>
    <li>Allows users to drop pin on the map</li>
    <li>Enter Remarks</li>
    <li>Address Fetching</li>
    <li>Pin Submission and saving</li>
    <li>Pin Navigation</li>
    <li>Store in Local Storage</li>
</ul>
